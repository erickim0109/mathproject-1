# 🌟 평면도형 탐험 실행 방법

이 저장소는 초등학생 대상의 인터랙티브 도형 학습 Streamlit 앱을 포함합니다.

## 준비

1. (선택) 가상환경 생성

```bash
python -m venv .venv
source .venv/bin/activate
```

2. 의존성 설치

```bash
pip install -r requirements.txt
```

3. 앱 실행

```bash
streamlit run app.py
```

브라우저가 자동으로 열리지 않으면 터미널에 표시된 주소(보통 `http://localhost:8501`)를 브라우저에 입력하세요.

---

문의사항이나 개선 요청은 리포지터리 이슈로 남겨주세요. 😀
